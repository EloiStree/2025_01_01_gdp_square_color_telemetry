# 2025_01_01_gdp_wow_pixel_position
This Godot package allows you to generate a pixel with a color that represents the position of a unit for color telemetry using the World of Warcraft coordinate system.


As you can see here, I was using it to export the map relative position of the player in a map with it rotation:
